
Objective: Showcase your CCST Cybersecurity knowledge.

Design: An edge router protecting an internal server farm.
Key Features:
Standard & Extended ACLs: Permit only specific traffic (e.g., HTTP/HTTPS) to the web server and block everything else.
NAT/PAT: Configure Static NAT for the server and PAT for internal users.
Port Security: Secure access switches to only allow specific MAC addresses, preventing unauthorized device plug-ins.
SSH Management: Disable Telnet and enforce SSH with local AAA.
